# How to Install Neutron
```
$ git clone https://github.com/Fahad-M/Neutron.git
$ cd Neutron
$ sudo chmod +x install.sh
$ ./install.sh
```

# How to Run
```
$ python3 neutron.py
```

# How it works
This program uses Mingw Cross compiler to compile the Code to Windows Executable. The uses Wine to run it.
